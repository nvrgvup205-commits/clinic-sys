## 👥 الأدوار والصلاحيات

### 1️⃣ **المالك (Owner)** 👑
```
✅ يدخل من: /owner
✅ يدير: كل العيادات في النظام
✅ يستطيع:
   - إنشاء عيادات جديدة
   - إنشاء أدمن لكل عيادة (من الواجهة!)
   - حذف عيادات
   - رؤية كل الإحصائيات
   - رؤية الحسابات الكاملة لكل العيادات
```

### 2️⃣ **الأدمن (Clinic Admin)** ⚙️
```
✅ يدخل من: /{clinic-slug}/staff
✅ صلاحيات في عيادته فقط:
   - إدارة المرضى
   - إدارة الأطباء
   - إدارة المواعيد
   - إدارة الخدمات والأسعار
   - إدارة شركات التأمين
   - إدارة موظفي الاستقبال والمحاسبين
   - رؤية كل التقارير والحسابات
   - إعدادات العيادة
```

### 3️⃣ **موظف الاستقبال (Receptionist)** 🏢
```
✅ يدخل من: /{clinic-slug}/staff
✅ صلاحيات محدودة:
   - تأكيد الحجوزات
   - استلام المدفوعات
   - إصدار الفواتير
   - إدارة الطوارئ
   - الرد على الشكاوى/الاستفسارات
   - رؤية المواعيد اليومية
❌ ما يستطيع:
   - تعديل بيانات الأطباء
   - تعديل أسعار الخدمات
   - رؤية التقارير المالية الكاملة
   
📍 ميزات خاصة:
   ✅ Clock In/Out (تسجيل وقت دخول وخروج)
   ✅ يوزر مميز لكل موظف
   ✅ كل فاتورة عليها اسم الموظف اللي عملها
```

### 4️⃣ **المحاسب (Accountant)** 📊
```
✅ يدخل من: /{clinic-slug}/staff
✅ صلاحيات محدودة جداً:
   - الدخول على لوحة الحسابات والتقارير فقط
   - رؤية كل التقارير المالية
   - تصدير التقارير (PDF, Excel, CSV)
❌ ما يستطيع:
   - رؤية الملفات الطبية
   - تعديل بيانات المرضى
   - أي عمليات إدارية
   
📍 ميزات خاصة:
   ✅ يوزر وباسورد لكل محاسب
```

### 5️⃣ **الدكتور (Doctor)** 👨‍⚕️
```
✅ يدخل من: /{clinic-slug}/staff
✅ صلاحيات:
   - رؤية مرضاه
   - شاشة الكشف (تشخيص + وصفة + فاتورة)
   - تأجيل أو إلغاء مواعيده
   - إدارة جدول عمله
   - رفع صور الأشعة
```

### 6️⃣ **المريض (Patient)** 👤
```
✅ يدخل من: /{clinic-slug}
✅ صلاحيات:
   - تسجيل / دخول
   - حجز موعد
   - تأكيد / إلغاء موعد
   - تقديم شكوى
   - طلب طارئ
   - رؤية تقاريره وفواتيره
   - تعديل بياناته
```

## 📦 المراحل والميزات

### ✅ **المرحلة 1: شركات التأمين**

#### الحالة: 🟢 جاهز للتنفيذ (SQL + Code جاهزين)

#### المميزات:
```
✅ إدارة شركات التأمين (10 شركات سعودية جاهزة)
✅ أسعار خاصة لكل شركة
✅ المرضى مرتبطين بشركة تأمينهم
✅ حساب تلقائي للخصومات
✅ زر تفعيل/إيقاف نظام التأمين من إعدادات العيادة
```

---

### 👑 **المرحلة 2: لوحة المالك المتطورة**

#### المميزات:
```
✅ إنشاء عيادة جديدة (موجود)
✅ إنشاء أدمن لكل عيادة من الواجهة (جديد!)
   - بدون الحاجة للدخول على DB
   - يحدد: username + password + full_name
   - يحدد الصلاحيات
✅ إدارة المستخدمين لكل عيادة
✅ صلاحية رؤية حسابات كل العيادات
```

---

### 💰 **المرحلة 3: أسعار الخدمات الكاملة**

#### كل خدمة لها:
```
✅ السعر العادي (Original Price)         → مثال: 200 ر.س
✅ سعر التأمين (Insurance Price)         → مثال: 180 ر.س
✅ نسبة الخصم (Discount %)               → مثال: 10%
✅ المدفوع بعد الخصم (Net Payable)       → مثال: 180 ر.س
```

#### تظهر في:
```
✅ لوحة الأدمن (للعرض والتعديل)
✅ لوحة الاستقبال (للتنفيذ)
✅ صفحة المريض (في الحجز)
```

---

### 🏢 **المرحلة 4: شاشة الاستقبال الكاملة**

#### الميزات:
```
✅ داشبورد موحد:
   - إحصائيات اليوم
   - الإشعارات الجديدة
   - مواعيد اليوم
   - الطوارئ
   - الشكاوى

✅ تأكيد الحجوزات (Pending → Confirmed)

✅ استلام المدفوعات:
   - نقدي / بطاقة / تحويل
   - اختيار الخدمة
   - تطبيق التأمين تلقائياً
   - زر "مدفوع" → الدكتور يستقبل المريض فوراً
   - الفاتورة محسوب عليها اسم الموظف + الوقت

✅ إدارة الطوارئ:
   - استقبال طلبات من المرضى (زر طوارئ)
   - الموظف يدخلها يدوياً
   - تحويلها لموعد

✅ الشكاوى والاستفسارات:
   - الرد عليها
   - متابعة الحالة

✅ نظام إشعارات Realtime:
   - حجز جديد
   - طوارئ
   - شكوى
   - أي تحديث

📍 ميزات Audit:
   ✅ Clock In/Out:
      - تسجيل وقت دخول الموظف
      - تسجيل وقت خروجه
      - حساب ساعات العمل
   ✅ كل فاتورة عليها:
      - اسم الموظف اللي عملها
      - الوقت
      - التاريخ
   ✅ الإدارة تشوف:
      - مواعيد عمل كل موظف
      - عدد الفواتير لكل موظف
      - الإنتاجية اليومية
```

### 📊 **المرحلة 11: الداشبورد المحاسبي والتقارير**

#### الصلاحيات:
```
✅ يدخل عليه: المحاسب + الأدمن + المالك
❌ لا يدخل عليه: الدكتور + الاستقبال + المريض
```

#### التقارير المطلوبة:

##### 1️⃣ **تقارير زمنية:**
```
✅ تقرير يومي
   - إجمالي الإيراد
   - عدد المرضى
   - عدد المواعيد
   - المدفوع cash/بطاقة/تأمين
   - الخدمات المقدمة
   - مصاريف اليوم
   - صافي الربح

✅ تقرير أسبوعي
   - كل اللي فوق +
   - مقارنة بالأسبوع السابق
   - أعلى يوم في الإيراد
   - ساعات الذروة

✅ تقرير شهري
   - كل اللي فوق +
   - تحليل شهري كامل
   - مقارنة بالشهر السابق
   - توقعات للشهر القادم
   - تقرير مفصل لكل دكتور
```

##### 2️⃣ **تقارير الإيرادات:**
```
✅ حسب طريقة الدفع
✅ حسب الخدمة
✅ حسب الدكتور
✅ حسب شركة التأمين
✅ حسب الوقت (صباح/ظهر/مساء)
✅ حسب يوم الأسبوع
```

##### 3️⃣ **تقارير المرضى:**
```
✅ مرضى جدد vs قدامى
✅ نسبة الاحتفاظ (Retention)
✅ متوسط الزيارات
✅ معدل التحويل
✅ معدل الإلغاء
✅ معدل عدم الحضور
```

##### 4️⃣ **تقارير الأطباء:**
```
✅ عدد المرضى لكل دكتور
✅ الإيراد لكل دكتور
✅ متوسط مدة الكشف
✅ تقييم المرضى
✅ ساعات العمل الفعلية
✅ معدل الإنتاجية
```

##### 5️⃣ **تقارير الموظفين:**
```
✅ لكل موظف استقبال:
   - وقت الدخول/الخروج
   - ساعات العمل
   - عدد الفواتير
   - إجمالي المبالغ المحصلة
   - معدل الإنتاجية
```

##### 6️⃣ **تقارير التأمين:**
```
✅ مطالبات معتمدة vs مرفوضة
✅ متوسط وقت السداد لكل شركة
✅ المبلغ المعلق
✅ تحليل لكل شركة
```

##### 7️⃣ **تقارير المصاريف:**
```
✅ مصاريف الأدوية والمواد
✅ مصاريف المختبر
✅ الرواتب
✅ الإيجار والمرافق
✅ صافي الربح الفعلي
```

#### الرسوم البيانية:
```
✅ Line Chart - تطور الإيراد
✅ Bar Chart - مقارنة الأيام/الأسابيع
✅ Pie Chart - نسب طرق الدفع/الخدمات
✅ Heatmap - أوقات الذروة
✅ Area Chart - النمو الشهري
✅ Funnel Chart - من حجز → دفع → كشف
```

#### التصدير والإرسال:
```
✅ تصدير PDF (للطباعة)
✅ تصدير Excel (للمحاسب)
✅ تصدير CSV (للتحليلات)
✅ تصدير صورة (للمشاركة)

✅ إرسال تلقائي بالإيميل:
   - تقرير يومي للمدير
   - تقرير أسبوعي للمالك
   - تقرير شهري شامل

✅ تخصيص:
   - فترة زمنية مخصصة
   - فلترة حسب دكتور/خدمة/تأمين
   - مقارنة فترتين
```
